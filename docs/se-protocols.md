# Shared Documentation Boundary

Shared documentation does not define product protocols or crate-specific message contracts.

Protocol models, protocol compatibility rules, generated protocol identities, and protocol-derived APIs belong to the crate-local documentation for the crate that owns or implements them.

Current protocol-related baselines are crate-local:

- `crates/mpi-core/se-design-baseline.md`
- `crates/mpi-macros/se-design-baseline.md`
